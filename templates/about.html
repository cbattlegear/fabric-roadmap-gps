About Fabric GPS

Pure, unfiltered roadmap data. We tap directly into the Official Microsoft Fabric Roadmap — reading the exact same JSON that powers Microsoft's site — so you're always seeing the source of truth.

Historical tracking that remembers everything. Since August 27, 2025, we've been watching every edit, every status flip, and every new feature announcement. Our Azure SQL Database captures it all, giving you a complete timeline of what changed, when it changed, and how it evolved.

AI-powered blog connections. We use Azure OpenAI embeddings and cosine similarity to automatically match releases with related articles from the Microsoft Fabric Blog. The semantic matching is pretty smart, but occasionally makes creative connections — treat it as a helpful suggestion rather than gospel. You can read more about how we built that over on Github.

Built in the open, powered by feedback. Have ideas? Found a bug? Drop us a line at cameron@fabric-gps.com or open an issue on GitHub. This tool exists because the community asked for it — your input shapes where it goes next.

Why Build Fabric GPS?

After several people in the span of a week asked for an easier way to track changes across the Microsoft Fabric roadmap, we built Fabric GPS to provide a simple and efficient solution.

Our goal is to make it easier for everyone to stay informed about the latest developments in the Microsoft Fabric ecosystem.
